# PORTFOLIO

This is a portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
